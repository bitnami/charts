# Bitnami Secure Images Helm chart for External Secrets

External Secrets Operator syncs secrets from external providers like AWS or Vault into Kubernetes Secrets automatically.

[Overview of External Secrets](https://external-secrets.io)

This Helm chart is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
