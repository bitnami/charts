# Bitnami Secure Images Helm chart for Rook Ceph Operator

The Rook Ceph Operator deploys and manages Ceph storage on Kubernetes when using Rook, providing block, object, and file storage on the cluster.

[Overview of Rook](https://rook.io)

This Helm chart is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
