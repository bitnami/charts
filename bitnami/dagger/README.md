# Bitnami Secure Images Helm chart for Dagger

Dagger is a programmable automation engine for building, testing, and shipping software. It runs pipelines in containers with caching and OpenTelemetry visibility across local and CI environments.

[Overview of Dagger](https://dagger.io)

This Helm chart is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
