# Bitnami Secure Images Helm chart for Ceph CSI Operator

The Ceph CSI Operator deploys and manages Ceph-CSI drivers (CephFS, RBD, and NFS) using CRDs, and is the recommended deployment mechanism for Ceph-CSI.

[Ceph CSI Operator project](https://github.com/ceph/ceph-csi-operator)

This Helm chart is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
