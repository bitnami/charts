# Bitnami Secure Images Helm chart for Crossplane

Crossplane is a framework for building without needing to write code cloud native control planes that can orchestrate applications and infrastructure no matter where they run.

[Overview of Crossplane](https://www.crossplane.io/)

This Helm chart is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
