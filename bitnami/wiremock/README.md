# Bitnami Secure Images Helm chart for WireMock

WireMock is an open-source tool for simulating HTTP-based APIs. It lets you mock external services to isolate your app during development and testing, featuring request matching, response stubbing, record/playback, and fault injection.

[Overview of WireMock](https://wiremock.org/)

This Helm chart is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
