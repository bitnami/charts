# Bitnami Secure Images Helm chart for OpenSearch Kubernetes Operator

OpenSearch Kubernetes Operator is a Kubernetes operator for automating OpenSearch cluster management using custom resources. It supports lifecycle management, rolling upgrades, TLS, and plugin installation.

[Overview of OpenSearch Kubernetes Operator](https://github.com/opensearch-project/opensearch-k8s-operator)

This Helm chart is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
